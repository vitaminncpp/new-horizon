# LearnSphere - Web-Based Interactive Learning Platform PRD

## 1. DESIGN INTENT
Modern, dashboard-style interactive learning platform.
- **Visuals:** Dark modern dashboard, rounded cards (16px), gradients, modular widget layout.
- **Roles:** Students, Instructors, Admins.

## 2. GLOBAL UI STYLE SYSTEM
- **Themes:** Dark (Primary), Light (Secondary).
- **Primary Gradient:** #6C5CE7 → #8E44AD (Purple)
- **Accent Gradient:** #00C9A7 → #4D96FF
- **Typography:** Inter (Primary). Heading XL (32px), Body (16px).
- **Corner Radius:** Cards 16px, Buttons 12px, Inputs 10px.

## 3. CORE UI LAYOUT STRUCTURE
- **Layout:** Dashboard shell with Left Sidebar (260px), Top Nav (72px), and Main Content Area.
- **Sidebar Items:** Dashboard, My Courses, Explore, Learning Path, Assignments, Quizzes, etc.
- **Top Bar:** Search, Notifications, Messages, Theme Toggle, Profile.

## 4. CORE UI COMPONENT LIBRARY
- **Buttons:** Primary, Secondary, Ghost, Icon, Danger.
- **Cards:** Course, Module, Lesson, Progress, Statistic, Notification.
- **Inputs:** Text, Password, Dropdown, Toggle, etc.
- **Learning-Specific:** Video Player, Code Editor, Quiz Panel, Notes Panel.

## 5. MVP PAGE ARCHITECTURE
1. **DASHBOARD (Student):** Stats, Continue Learning, Recommended, Activity.
2. **COURSE EXPLORER:** Search, Filters, Course Grid.
3. **COURSE DETAILS:** Module List, Lesson Preview, Instructor Info.
4. **LEARNING PLAYER:** Sidebar Modules, Center Lesson (Video/Code/Quiz), Right Notes/Discussion.
5. **QUIZ INTERFACE:** Timer, Question Nav, Feedback, Result Screen.
6. **ASSIGNMENTS:** List, Submission Panel.
7. **DISCUSSIONS:** Thread-based layout.
8. **USER PROFILE:** Progress, Certificates, Achievements.
9. **PROGRESS ANALYTICS:** Charts (Completion, Time Spent).
10. **CERTIFICATES:** Grid layout.
11. **SETTINGS:** Profile, Security, Notifications.
12. **ADMIN PANEL:** User/Course management.

## 12. BRANDING
- **Platform Name:** LearnSphere
- **Tagline:** "Learn. Practice. Master."
